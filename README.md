# Coding Test Preparation 🧑‍💻

A collection of Python solutions to common coding interview and technical-assessment questions — covering searching, sorting, arrays, strings, and number problems. Built as a personal practice repo while preparing for coding tests and technical interviews.

## 📌 About

This repository contains short, self-contained Python scripts that solve classic coding problems frequently asked in interviews and online assessments. Each file focuses on a single problem, making it easy to review, run, and reuse.

## 📂 Contents

| File | Problem |
|------|---------|
| `2nd_largest.py` | Find the second largest number in a list |
| `Two_sum.py` | Find two numbers in an array that add up to a target |
| `binary_search.py` | Search a sorted array using binary search |
| `bubble_sort.py` | Sort an array using the bubble sort algorithm |
| `duplicates.py` | Find duplicate elements in a list |
| `even_odd_count.py` | Count even and odd numbers in a list |
| `fizz_buzz.py` | Classic FizzBuzz problem |
| `freq_count.py` | Count frequency of elements in a list |
| `frequency.py` | Frequency analysis of elements/characters |
| `linear_search.py` | Search an array using linear search |
| `max_min.py` | Find the maximum and minimum values in a list |

*(List reflects files as of the latest commit — check the repo for the most up-to-date set.)*

## 🚀 Getting Started

### Prerequisites
- Python 3.x installed on your machine

### Running a script
Clone the repo and run any file directly:

```bash
git clone https://github.com/aditya-datahub/coding-test-preparation.git
cd coding-test-preparation
python <filename>.py
```

For example:
```bash
python binary_search.py
```
